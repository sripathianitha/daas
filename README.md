# daas

A Django web application for helping debug and analysis automation run failures


### Prerequisites

```
Python 3.10
Django 4.2
```

### Installing

1. Download/Clone the repository.
2. Open terminal/command and navigate to downloaded folder(daas) using 'cd' command
3. Then Install dependencies using requirement.txt file by below command

```
pip install -r requirement.txt
```
4. After completion of pip install, enter the below command

```
python manage.py runserver 0.0.0.0:8000
```
5. Then open browser and enter below url
```
http://localhost:8000/
```
